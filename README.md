# Graph_Neural_ODE_Sudirman_Thamrin
Sistem ini menggunakan arsitektur Graph Neural Ordinary Differential Equations (Graph Neural ODE) untuk memodelkan dan memprediksi dinamika lalu lintas di sepanjang jalur utama Sudirman hingga Thamrin, Jakarta. Model ini menangani data spasial (tata letak jalan) sekaligus data temporal (perubahan waktu).
